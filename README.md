 ## Objectifs de l'application:
 Utiliser des structures de données.</br>
->	Utiliser les fichiers.</br>
-> Un joueur doit créer un compte avant de commencer le jeu ou se connecter s’il y a un compte.</br>
-> Chaque partie se joue entre 2 joueurs soit (joueur contre un autre joueur ou bien joueur contre un ordinateur).</br>
->	Afficher l'historique de joueur courant.</br>
->	Afficher les dix meilleurs scores des joueurs. </br>
-> utilise L’algorithme Minimax .</br>
